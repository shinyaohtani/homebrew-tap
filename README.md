# shinyaohtani/homebrew-tap

Homebrew tap for [shinyaohtani](https://github.com/shinyaohtani)'s macOS apps.

## Available Casks

- **linkkeeper** — [LinkKeeper](https://github.com/shinyaohtani/link-keeper): macOS native bookmark manager utility

## Usage

```bash
brew install --cask shinyaohtani/tap/linkkeeper
```

## Updating

```bash
brew update
brew upgrade --cask linkkeeper
```

## Uninstalling

```bash
brew uninstall --cask linkkeeper

# Remove all data including bookmarks
brew uninstall --zap --cask linkkeeper
```
